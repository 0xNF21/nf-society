2. **Bandeau** : le bandeau ambre "Mode Demo" s'affiche en haut
3. **Morpion** : creer une partie → slug DEMO-XXX → jeu vs bot fonctionnel
4. **Memory** : creer une partie → slug DEMO-difficulty-XXX → jeu solo fonctionnel
5. **Daily** : scratch et spin fonctionnent sans paiement
6. **Shop** : achat avec XP fonctionne
7. **Profil** : affiche les stats demo (XP, level, streak)
8. **XP** : verifier que les gains XP s'affichent apres chaque action
9. **Persistance** : recharger la page → demo toujours actif
10. **Quitter** : bouton "Quitter" desactive le mode, bandeau disparait

Pour chaque point, utilise le preview server pour verifier visuellement.
